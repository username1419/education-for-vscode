Lesson 2: Functions

Functions are resuable sets of instructions that the computer follows to complete a task to produce a result. Sometimes, you may provide the computer with an input to give it information on what you want it to create.
A function in Python is created like this: 

def my_function():
    print("Running my_function")

The combination of letters after def specifies the name of the function, and the lines which have indentation after def specifies the instructions that the function does when called. In this example, the function is named my_function, and running it makes the computer execute print("Running my_function"). To make the computer run the function, also known as calling it, below the function creation we write: 

my_function()

Running the file like this will make the computer display: 

Running my_function

Functions can have 1 or more inputs, called arguments. This can be used inside the function like so: 

def product(num1, num2):
    print(num1 * num2)

The function takes 2 arguments: num1 and num2. Using this function, calculating the product of 3 and 10 is as simple as writing: 

product(10, 3)

Running this displays: 

30

The return keyword tells the computer to stop the program at that point in the function and return to the function call and replace the function with the return value. For example, rewriting product(num1, num2): 

def product(num1, num2):
    return num1 * num2

This means that calling product(10, 3) no longer displays 30:

However, this also means that we can use this to assign a variable the result of the function call: 

result = product(10, 3)
print(result / 2)

Doing this will output: 

15
