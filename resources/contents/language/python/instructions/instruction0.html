Lesson 0: Data types

Computers, since their inception, has stored information as '1's and '0's, or binary. Data types are used by them to interpret these chains of binary information and process them. In Python, this information is separated into 6 basic data types.

Characters and Words in Python are written in single - 'a' , 'word' - or double quotes - "a", "word".
Truth values in Python are represented by True or False.
Whole numbers in Python are written as is - -1, 2
Real numbers with decimal points in Python are also written as is - 0.1, -0.2.
The absence of a value is represented in Python as the word None.
