Lesson 1: Variables, Outputs, and Inputs

Part One: Variables

Variables are containers for a value representing a specific data type.

Variables are created in 2 processes: Declaration and Value assignment.
Declaration is the first process of creating a variable. It involves naming the variable and assigning a data type to it.
Value assignment is the second process of creating a variable. It involves assigning a value to the previously declared variable. The to-be assigned value must match the data type of the variable it will be assigned to.
In Python, these 2 processes are done at the same time using the statement: variable = value. For example, this is the code to create a variable named message and assign the string value Hello! to it:

message = "Hello!"

Part Three: Inputs
